# Implementation Notes — Regen Community Evals

## Current Intent

This pack converts the stronger Gemma 4 Good framing into executable eval work. The point is not
"grant opportunities"; it is whether Coop can help regenerative communities round up scattered
knowledge, keep sensitive context private, and publish useful action briefs.

## Eval Shape

- Four group types by four action types.
- Minimum 32 cases: canonical plus stress/privacy/noise variant per matrix cell.
- Santa Ana Watershed is the land/watershed hero row.
- Every case produces or expects an action brief with public/private separation.
- Browser model-in-loop Gemma 4 proof is mandatory for final lane completion.
