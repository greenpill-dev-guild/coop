# Coop Audio Asset Manifest

Use this folder for source-of-truth audio assets and their provenance records.

## Required V1 Events

- `coop-rooster-call`
- `coop-soft-cluck`
- `coop-squeaky-test`

## Manifest Fields

For every externally sourced audio file added here, record:

- filename
- event
- source URL
- source library
- creator/uploader
- license
- attribution required: yes/no
- date added
- edits made
