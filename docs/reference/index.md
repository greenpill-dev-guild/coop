## How To Read Status Notes

Some reference pages preserve older plans, audits, and research because they still explain decisions
or constraints. When a page is marked historical or provisional, treat its local note as part of the
document: use the linked current-state docs when a detail conflicts with the live product surface.

The reference archive should stay useful without pretending every document has the same authority.
